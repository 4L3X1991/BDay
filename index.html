🛠 2. Webseite kostenlos online hosten

Wir nutzen GitHub Pages, weil es kostenlos und einfach ist. Falls du noch keinen GitHub-Account hast, erstelle einen unter github.com.

Schritt-für-Schritt-Anleitung:

  1. Gehe auf GitHub und logge dich ein.

  2. Erstelle ein neues Repository:

    • Klicke auf das “+” (oben rechts) > New repository.
    • Gib als Name z. B. “gutschein” ein.
    • Setze es auf Public.
    • Aktiviere "Add a README file".
    • Klicke auf "Create repository".

  3. Code hochladen:

    • Öffne dein Repository.
    • Klicke auf "Add file" > "Upload files".
    • Ziehe die HTML-Datei (die ich erstellt habe) hier rein oder wähle sie aus.
    • Klicke auf "Commit changes".

  4. GitHub Pages aktivieren:

    • Gehe zu Settings > Pages.
    • Wähle als Branch "main" und Ordner "/root".
    • Speichere die Einstellung.
    • Nach ein paar Minuten ist deine Seite unter https://DEIN-GITHUB-USERNAME.github.io/gutschein/ erreichbar!

📲 3. QR-Code generieren

  1. Gehe auf qr-code-generator.com.

  4. Drucke ihn aus und verschenke ihn. 🎁
